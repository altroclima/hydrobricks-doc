GSM-SOCONT
----------

GSM-SOCONT is a conceptual glacio-hydrological model described in Schaefli2005_.

Some basic properties are given in the following table.

   :widths: 50 50
   :header-rows: 0

   * - Spatial structure
     - semi-lumped (elevation bands)
   * - Time step
